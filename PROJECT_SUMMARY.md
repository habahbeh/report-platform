## 🏗️ البنية المنطقية

### المفاهيم الأساسية

```
Template (قالب)
    ├── Axis (محور) — مثال: "البحث العلمي"
    │       ├── Item (بند/مؤشر) — مثال: "3.1 عدد الأبحاث في SCOPUS"
    │       ├── Item — مثال: "3.2 معدل الاستشهاد"
    │       └── Item — مثال: "3.3 براءات الاختراع"
    │
    └── Entity (جهة مسؤولة) — مثال: "عمادة البحث العلمي"
            └── مسؤولة عن تقديم بيانات بنود معينة
```

```
┌─────────────────────────────────────────────────────────────┐

                              ↓
┌─────────────────────────────────────────────────────────────┐
│  2. جمع البيانات                                             │
│     ├── إرسال روابط للجهات المسؤولة                         │

│     ├── رفع ملفات Excel عند الحاجة                          │
│     └── متابعة التقدم وإرسال تذكيرات                        │

                              ↓
┌─────────────────────────────────────────────────────────────┐
│  3. مراجعة البيانات                                          │
│     ├── التحقق من صحة المدخلات                              │

│     └── طلب تعديلات من الجهات                               │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  4. التوليد بالذكاء الاصطناعي                                │
│     ├── AI يكتب النصوص التحليلية لكل بند                    │

└─────────────────────────────────────────────────────────────┘
```

## 📊 هيكل البيانات

### القالب (Template)
```
التقرير السنوي للجامعات الأردنية
├── المحور 1: الاعتمادية وضمان الجودة (10 بنود)
├── المحور 2: التدريس (7 بنود)
├── المحور 3: البحث العلمي (8 بنود)
├── المحور 4: إدارة الموارد البشرية والمالية (14 بند)
├── المحور 5: البنية التحتية (5 بنود)
└── المحور 6: خدمة المجتمع (5 بنود)
```

### البند (Item)
كل بند يحتوي على:
| الحقل | الوصف | مثال |
|-------|-------|------|
| `code` | رمز البند | 3.1 |
| `name` | اسم البند | عدد الأبحاث في SCOPUS |
| `unit` | وحدة القياس | بحث |
| `field_type` | نوع البيانات | number / percentage / text |
| `current_value` | القيمة الحالية | 150 |
| `previous_value` | القيمة السابقة | 120 |
| `change_percentage` | نسبة التغير | +25% |

### المسودة (Draft)
لكل بند مسودة تحتوي على:
- **content**: النص التحليلي المولّد بالـ AI
- **table_data**: بيانات الجدول (JSON)
- **chart_config**: إعدادات الرسم البياني
- **manual_content**: محتوى يدوي (صور، جداول إضافية)
- **status**: حالة المسودة (not_started → generated → edited → approved)

### الخدمات الرئيسية

#### 1. Generation Service
```python
# توليد المحتوى بالذكاء الاصطناعي
service = GenerationService(period, model='cli')
result = service.generate_axes(axis_ids=[1, 2, 3])
result = service.generate_items(item_ids=[1, 2, 3])
```

#### 2. Chart Service
```python
# توليد الرسوم البيانية
service = ChartService()
chart = service.generate_bar_chart(data, title='...')
chart = service.generate_pie_chart(data, title='...')
chart = service.generate_comparison_chart(current, previous, label)
```

#### 3. Export Service
```python
# تصدير التقرير إلى Word
service = ExportService(period)
output = service.export_to_word(
    include_items=True,
    include_charts=True,
    include_tables=True,
    approved_only=False
)
```

#### 4. Data Import Service
```python
# استيراد البيانات من ملفات Excel
result = import_all_data(period_id=1, data_folder='path/to/data')
```

## 🔧 كيفية الاستخدام

### 1. تشغيل السيرفر
```bash
# Backend
cd report-platform/backend
source venv/bin/activate
python manage.py runserver 8002

# Frontend
cd report-platform/frontend
npm run dev
```

### 2. الوصول للمنصة
- **Dashboard:** http://localhost:3000/dashboard
- **API:** http://localhost:8002/api/

### 3. توليد التقرير
1. اذهب إلى `/dashboard/generate`
2. اختر الفترة
3. اضغط "توليد الكل" للمحاور
4. اضغط "توليد البنود" للتفاصيل
5. راجع وعدّل المحتوى
6. اضغط "تصدير Word"
